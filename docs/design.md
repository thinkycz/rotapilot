# Design References

RotaPilot does not currently ship with a hand-off design source (Figma,
Google Stitch, etc.) checked into the repository. UI work to date has
been driven by the role flows in
`docs/specs/rotapilot-mvp.md` and the component sketches in
`resources/js/components/ui`.

When a design source becomes available, drop the asset (image, Figma
export, Stitch project) into this folder and add a row to the table
below so the design system and the implementation stay in lock-step.

| ------------ | ------------------------------------------------ | ------ |
